# Branding Assets

This folder contains high-resolution branding assets for Prunance.

## Files
- `logo_vector.svg`: Full logo (Prunance wordmark + symbol)
- `icon_vector.svg`: App icon symbol

Since these are SVG (Scalable Vector Graphics), you can scale them to any resolution (1024x1024, 2048x2048, etc.) without losing quality.

To export as high-res PNG:
1. Open the SVG in a browser or design tool (Figma, Inkscape).
2. Use the "Export" or "Save As" function to choose your desired PNG resolution.
